This is display:none

Display: none is a CSS property used to hide website elements. here the website word is making it invisible.

This is display:inline

The display option of inline tells the element to fit itself on the same line.

This is block

An element that has the display property set to starts on a new lineand takes up the available screen width.

This is inline-block

The display option of inline-block allows the element to fit itself on the same line, but it also allows other elements to sit next to it.
